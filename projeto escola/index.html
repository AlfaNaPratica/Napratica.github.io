Lojas de Departamento irão conceber cupons de desconto para Km's andados de bicicleta

Prefeitura libera cupons para incentivar o exercício físico

Com o alto índice de tráfego de carros, empresas e a prefeitura se unem para combater à essa questão que afeta a sociedade

Ao longo do tempo, as empresas estão achando métodos para tentar controlar o aquecimento global.

Fazendo isso chegaram a conclusão de criar uma promoção para conscientizar as pessoas, a pararem de usar os automóveis movidos derivados de petróleo, começando a utilizar as bicicletas.

A cada 8 km's andados, será concebido um cupom de 15% de desconto.
